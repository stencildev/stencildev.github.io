# About

I am a technical support engineer with a strong interest in infrastructure, systems administration, and DevOps-oriented work.

My background is rooted in troubleshooting complex systems and understanding how software behaves in real environments. Over time, that curiosity expanded beyond supporting applications into building and operating the infrastructure that runs them.

This led me to hands-on work with Linux systems, Docker-based service deployment, reverse proxies, DNS infrastructure, and homelab architecture.

This site serves as both a portfolio and a documentation space for selected projects, infrastructure experiments, and operational runbooks.
